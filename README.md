# Priestess Desktop

基于PyQt5和LangChain的桌面宠物项目，角色设定为《明日方舟》中的普瑞赛斯。

## 主要内容

1.  pet.py: 桌面宠物的启动程序。
  - 显示普瑞赛斯的图片（无边框，背景透明）。
  - 支持鼠标拖拽移动。
  - 自动吸附：靠近屏幕边缘（200px内）会自动吸附。
  - 聊天功能：双击或右键可打开对话窗口，窗口会自动跟随宠物位置。
    
2.  main.py: 核心 AI 逻辑。
  - PriestessAI 类负责处理对话。
  - 使用 RAG（检索增强生成）技术，基于本地数据库 (chroma_db) 回答问题。
  - 也可以单独运行此脚本在命令行中对话。

3.  ingest.py:用于将资料导入数据库的脚本。
